Themes

Updates

The good thing about spicetify is that it is actively been upgraded and hence you recieve all the latest updates and features Spotify is pushing

Spotify, every now and then, updates its client. Since we have no power over this process, you will likely need to re-apply Spicetify.

If there are major changes to spotify we will have to re-apply spicetify

spicetify upgrade

If no update for Spicetify is available, it means that it still works by simply running

spicetify backup apply

After running spicetify upgrade,(i.e updates were installed) you will be prompted to run

spicetify restore backup apply

you will have everything back where it was, your settings are saved!!
